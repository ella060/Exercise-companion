# 中老年运动陪伴 App

一个温暖的中老年女性居家健身应用，提供精神陪伴和简单健身指导。

## 功能特性

- 外部视频导入（支持微信、抖音、B站等链接）
- 每日打卡海报生成与分享
- 温暖推送（精神陪伴）
- 就医/康复待办记录

## 技术栈

- 前端: HTML5 + CSS3 + JavaScript
- 后端: Node.js + Express
- 数据库: SQLite

## 项目结构

```
.
├── app/                # 前端应用
│   ├── css/
│   ├── js/
│   └── index.html
└── backend/            # 后端服务
    ├── src/
    └── package.json
```

## 开发

### 前端
直接打开 `app/index.html` 即可预览

### 后端
```bash
cd backend
npm install
npm run dev
```
